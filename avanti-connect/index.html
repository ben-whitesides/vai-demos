Change 03 — Kill the State Wall
Search + collapsible regions
The 50-dot state wall is the biggest UX problem. Visual noise on a phone, hard to tap accurately, no information density. Replace with:
(a) Search bar at top — type tex, get Texas
(b) Region groups (West, Southeast, Midwest, Northeast) collapsed by default
(c) Per-region count (e.g. 3 / 13) so user sees their selection at a glance
(d) Top-right Nationwide shortcut — one tap selects all

Change 08 — AVANTI = Always Green
The word AVANTI is always rendered in AVANTI green
Brand rule: anywhere the word "AVANTI" appears in product copy, marketing, or UI, it's rendered in #34d399 at heavier weight. Treats the word like a logotype, not body text. Current screen uses default system font everywhere. Match the VAI brand: Oswald for the screen title, section labels, and CTA. JetBrains Mono for state codes (CA / TX / NY) and counts — gives the numeric data that surgical-instrument feel.
Change 06 — Subhead + Selection Echo
Tell the user what this screen does + what they've picked
Add a one-line subhead under the title: "Find college coaches recruiting your sport." Cold users need orientation in 1 sentence.

Above the CTA, show a tiny selection summary: 3 sports • 2 divisions • 3 states. Cheap reassurance that the filters are working.
Change 07 — CTA Polish
Sticky bottom + gradient fade
Current CTA is a flat orange bar. Refined version: solid gold with black uppercase Oswald text, sticky to bottom of screen with a soft fade behind it so list content scrolls naturally underneath. Feels premium, not utility.
