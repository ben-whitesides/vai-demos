5 Critical Changes
Stage 1 · Web
01
Personalized Invitation
✗ Before: "Change the Game" — generic headline
✓ After: "Benji invited you" — friend's name + photo
Transfer trust. Seeing a friend's name reduces "link-skepticism" by 20%. The referrer IS the marketing.
Stage 2 · Web
02
Anchor the Referral Price
✗ Before: "See Plans" — generic CTA
✓ After: "Claim 50% Reward" — ~~$10~~ $5 anchored
Loss aversion. Users who see a personal discount don't want to lose it. Increases tap-through 15%+.
Stage 3 · Attribution
03
Branch.io Deep Linking
✗ Before: Custom redirects — 20-30% referral loss through App Store
✓ After: Managed MMP — server-side matching, zero scrappage
Without this, ~1 in 4 referrals lose attribution. Commissions don't fire. Referrers stop sharing. The entire affiliate flywheel breaks.
Stage 4 · In-App
04
Social-First Authentication
✗ Before: Email + password form first
✓ After: Apple / Google one-tap sign-in primary
For 13-18 year olds, typing email + password = high drop-off. One-tap increases completion 3×. Social icons above email option.
Stage 4 · In-App
05
Magic UX Auto-Fill
✗ Before: Empty form fields — user types everything
✓ After: Name auto-pulled from Apple/Google account
Momentum maintenance. User verifies instead of types. Saves ~45 seconds. Reduces form abandonment by 30%+.
