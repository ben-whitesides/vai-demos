Design Rules for Claude Design
Home Grid Logic
Core tiles always visible: Feed, Groups, Schedule, Chat, Wallet, Share
PLAY Status + Passport → Me/Profile tab only
Association hero tile appears when user joins a club/camp
Club OS tiles appear in sub-section below core grid
Animate hero tile in with a gentle slide from top
Wallet Tile
Sub-text shows balance + referral count
Affiliate dashboard lives INSIDE Wallet (not behind Share button)
Share tile is separate — links directly to user's affiliate link
Share tile always shows EARN badge in green
Drill-In Navigation
Hero tile tap → new screen with club-specific sub-grid
Back navigation: "‹ Home" in header, never back button
AVANTI strip persists on drill-in screens, scoped to that context
Sub-grid tiles use same tile component — not a separate design
AVANTI Green
AVANTI ONLINE indicator: #34d399, pulsing dot
AVANTI strip label: #34d399, JetBrains Mono caps
AVANTI nav tab: green icon + label, subtle green bg
Confirm chips: orange bg, BLACK text (non-negotiable)
