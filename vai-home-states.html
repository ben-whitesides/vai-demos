Design Notes — Claude Design Feed
TILE LOGIC
• Standard tiles always visible (tier-gated)
• Club tiles only when user is member
• Camp tiles only when registered
• Divider labels show the association name
• Wide hero tile for active/live events
• Locked tiles show upgrade CTA on tap
AVANTI STRIP
• Quiet by default (State 1 — no issues)
• Shows when: RED PLAY, waiver deadline, camp action
• Yellow = confirm chip (black text, orange bg)
• Dismiss stores 24h preference
• AVANTI label in --vai-green #34d399
COACH DIFFERENCE
• Roster is tile #1 (not Schedule)
• PLAY Status shows team aggregate
• AVANTI strip = team-level alerts
• No Wallet/Bookings/Wearables by default
• QA chip row reorders to: Roster, PLAY, Schedule
DESIGN TOKENS
• Page: #060608 · Card: #0c0c10
• Orange: #f7941e · Green: #34d399
• Font: Oswald 600 (labels) + DM Sans (body)
• Icons: SVG 1.5px stroke, rounded
• Black text on all colored buttons
